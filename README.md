# Recipes 
A collection of projects and code recipes showcasing experiments, patterns, and techniques

## Available Recipes

- [**astro-cloudflare-env-context**](./astro-cloudflare-env-context) — Custom context wrapper for accessing Cloudflare runtime environment variables in nested functions with Astro SSR.
- [**astro-cloudflare-env-middleware**](./astro-cloudflare-env-middleware) — Middleware with AsyncLocalStorage to access Cloudflare runtime environment variables throughout Astro SSR code.
- [**astro-cloudflare-hono-openapi**](./astro-cloudflare-hono-openapi) — Example of an Astro SSR project on Cloudflare with an API endpoint extended using Hono and OpenAPI.
